# Ressources projet (code source)

Contenu livré avec l'application, qui n'appartient pas aux données utilisateur (`data/`).

- **guides/** : pages HTML pour guider la création des clés API (Airtable, Claude Code).
- **examples/** : exemples JavaScript issus du POC, utiles pour coder des plugins (extraction d'offres depuis emails, etc.).

Ces fichiers sont versionnés et ne doivent pas être placés dans `data/` (répertoire des données utilisateur).
