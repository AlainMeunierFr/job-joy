# 1. Créer un compte AirTable
1. Rendez-vous sur [AirTable](https://airtable.com/).
2. Cliquez sur **"Sign up"** pour créer un compte.
3. Remplissez les informations nécessaires (email, mot de passe, etc.).
4. Validez votre compte via le lien envoyé par email.

# 2. Générer une clé API AirTable
1. Une fois connecté, allez dans votre [compte AirTable](https://airtable.com/account).
2. Dans la section **"API"**, cliquez sur **"Generate API key"**.
3. Copiez la clé API générée et copiez là dans le formulaire de l'app

# Remarques
- Assurez-vous de **ne pas partager** votre clé API publiquement.
