1. Créer un compte Claude Code

  • Rendez-vous sur Console Anthropic.
  • Cliquez sur Sign up pour créer un compte.
  • Remplissez les informations nécessaires (email, mot de passe, etc.).
  • Validez votre compte via le lien envoyé par email.
  • Une fois connecté, accédez à votre Dashboard.

2. Acheter des crédits

  • Dans la console, cliquez sur Billing (Facturation).
  • Ajoutez un moyen de paiement (carte bancaire).
  • Sélectionnez le montant des crédits à acheter.
  • Validez le paiement.
  • Vérifiez que vos crédits sont bien visibles dans votre espace Billing.

3. Créer une clé API (API Key)

  • Dans la console, allez dans la section API Keys.
  • Cliquez sur Create Key.
  • Donnez un nom à votre clé API.
  • Cliquez sur Create.
  • Une fenêtre s'ouvre avec votre clé.
  • Cliquez sur l'icône pour copier la clé API.
  • Collez cette API Key dans l'application.
