1. Créer un compte Mistral AI

  • Rendez-vous sur Console Mistral AI.
  • Cliquez sur Sign up pour créer un compte.
  • Remplissez les informations nécessaires (email, mot de passe, etc.).
  • Validez votre compte via le lien envoyé par email.
  • Une fois connecté, accédez à votre Dashboard.

2. Activer l’accès API

  • Dans la console, vérifiez que l’accès à l’API est activé pour votre compte.
  • Selon votre plan, vous pouvez bénéficier d’un crédit gratuit de test.
  • Si nécessaire, ajoutez un moyen de paiement dans la section Billing.
  • Vérifiez que votre compte est bien actif pour effectuer des requêtes API.

3. Créer une clé API (API Key)

  • Dans la console, allez dans la section API Keys.
  • Cliquez sur Create new key.
  • Donnez un nom à votre clé API (ex : MonApplication).
  • Cliquez sur Create.
  • Une fenêtre s’affiche avec votre clé API.
  • Copiez immédiatement la clé (elle ne sera plus affichée ensuite).
  • Collez cette API Key dans l'application.
