1. Créer un compte Airtable

  • Rendez-vous sur Airtable.
  • Cliquez sur Sign up pour créer un compte.
  • Remplissez les informations nécessaires (email, mot de passe, etc.).
  • Validez votre compte via le lien envoyé par email.

2. Créer la base Airtable

  • Dans Airtable, créez une nouvelle base (ou ouvrez une base existante).
  • Ouvrez la base et copiez l'URL complète de la base dans la barre d'adresse. Exemple : https://airtable.com/appMpMmSQSlL1UARS/…
  • Collez cette URL dans l'application.

3. Créer une clé API (API key)

  • Une fois connecté, allez dans la création de Jetons d'accès personnels.
  • Cliquez en haut à droite sur "Créer un jeton"
  • Donnez lui un nom
  • Dans "champs d'application" sélectionnez
  • Dans "accès", "Ajouter une base", sélectionnez la base de données créé en 2.
    • data.records:read
    • data.records:write
    • schema.bases:read
    • schema.bases:write
  • Cliquez sur "Créer le jeton" : une fenetre s'ouvre
  • Cliquez sur l'icône de presse papier pourcopier l'APi Key
  • Collez cette API Key dans l'application.
