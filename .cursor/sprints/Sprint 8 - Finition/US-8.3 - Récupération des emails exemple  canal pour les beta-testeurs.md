# US-8.3 — Récupération des emails exemple : canal pour les beta-testeurs

Moyen pour les utilisateurs (beta-testeurs) de t’envoyer des emails exemple afin que tu puisses développer un nouveau plugin (ex. nouveau site d’offres). Le plugin sera diffusé ensuite par mise à jour. Canal à définir (export + envoi, formulaire avec pièce jointe, etc.).
